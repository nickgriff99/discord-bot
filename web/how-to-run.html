Setup

Run the bot and add it to your server

Discord application, permissions, .env, then npm start. The terminal prints the URL for web/ (default http://localhost:4173; optional WEB_PORT).

Discord application

Create the bot and invite it

  1. Open the Discord Developer Portal.
  2. Create a new application.
  3. In Bot, add a bot user and copy the token for DISCORD_BOT_TOKEN.
  4. In General Information, copy the Application ID for DISCORD_CLIENT_ID.
  5. In OAuth2 > URL Generator:
    • Scopes: bot, applications.commands
    • Bot permissions: Send Messages, Use Slash Commands, Connect, Speak, Use Voice Activity
  6. Open the generated invite URL and add the bot to your server.

YouTube Data API

API key for search and playback

  1. Open Google Cloud Console.
  2. Create or select a project.
  3. Enable YouTube Data API v3.
  4. Create an API key and set YOUTUBE_API_KEY in your .env.

Environment

Create .env in the project root

DISCORD_BOT_TOKEN=your_discord_bot_token
DISCORD_CLIENT_ID=your_discord_client_id
YOUTUBE_API_KEY=your_youtube_api_key
# Optional: register slash commands to one guild for instant updates while testing
# DISCORD_GUILD_ID=your_test_server_id

If commands do not appear immediately, set DISCORD_GUILD_ID for your test server or wait for global registration to propagate.

Start

Install dependencies and run

1. Install

npm install

2. Start bot + web

npm start

3. Dev (watch)

npm run dev

4. Web only

npm run web:start

After npm start, the terminal shows the web/ URL. Optional: WEB_PORT. Hosting: HOSTING.md.
